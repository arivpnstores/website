API ENDPOINTS PEMBAYARAN

Dokumentasi endpoint untuk integrasi pembayaran QRIS

GET /orderkuota/createpayment

Buat Pembayaran QRIS

Parameter Query:

apikey: [YOUR_API_KEY]
amount: [jumlah pembayaran]
codeqr: [kode QR unik]

Contoh Request:

axios.get(`https://api.serverpremium.web.id/orderkuota/createpayment?apikey=[YOUR_API_KEY]&amount=${amount}&codeqr=${UrlQr}`);
GET /orderkuota/cekstatus

Cek Status Pembayaran

Parameter Query:

apikey: [YOUR_API_KEY]
merchant: [merchant ID]
keyorkut: [API key order kuota]

Contoh Request:

axios.get(`https://api.serverpremium.web.id/orderkuota/cekstatus?apikey=[YOUR_API_KEY]&merchant=${global.merchantIdOrderKuota}&keyorkut=${global.apiOrderKuota}`);
